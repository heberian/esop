# esop
ESOP API

SDK Requirements

JDK 1.8 or above

1) Check out this repository where ever you want

2) Open Command Prompt or Terminal in your computer

3) Go to the repository path

4) For example your check out was in C. Then go to  C:\esop>

5) Then run below command

C:\esop> java -jar esop.jar

6) Press ENTER
